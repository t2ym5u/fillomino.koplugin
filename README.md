# fillomino.koplugin

A Fillomino puzzle plugin for [KOReader](https://github.com/koreader/koreader).

## Rules

Fill every cell so that each group of orthogonally connected cells sharing the same number has exactly that many cells. Two same-number groups may not touch orthogonally. Given clue numbers are fixed.

## Features

- **Multiple grid sizes**
- **Three difficulty levels** — Easy, Medium, Hard
- **Check** — highlights constraint violations
- **Auto-save** — puzzle state saved and restored on next launch

## Installation

1. Download `fillomino.koplugin.zip` from the [latest release](../../releases/latest).
2. Extract into the `plugins/` folder of your KOReader data directory.
3. Restart KOReader.
4. Open the menu → **Tools** → **Fillomino**.

## Controls

| Action | How |
|--------|-----|
| Select a cell | Tap it |
| Enter a number | Tap the digit button |
| Erase a cell | Tap **Erase** |
| Check progress | Tap **Check** |
| New puzzle | Tap **New** |
| Show rules | Tap **Rules** |
